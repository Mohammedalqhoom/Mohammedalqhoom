A desktop management system for tailoring shops built with **C#**, **.NET Framework**, **Windows Forms**, **SQL Server**, and **Crystal Reports**.

- Order management with status tracking
- Customer records and measurement history
- Fabric and inventory management
- Purchase invoices and supplier costs
- User authentication and role-based access control
- 3-Tier Architecture: UI, BLL, DAL

Repository: [Tailoring Management System MTS](https://github.com/MOGAHEDMOHAMMED/Tailoring-Management-System_MTS)

- Built with Flutter, Dart, SQLite, Firebase, and GetX
- Offline-first workflow with local database structure
- Inventory, cashier, debts, invoices, and reporting screens
- Designed for real store operations and business automation

More details: [Portfolio](https://mohammedalqhoom.github.io/my-portfolio/)
