## Professional Summary

I am a software engineer focused on building **business systems** and **data-driven desktop applications** using **C#**, **.NET Framework**, **Windows Forms**, and **SQL Server**.

My work starts before writing code: I focus on understanding the business problem, analyzing requirements, designing the system structure, and building a database that supports real workflows. I am especially interested in ERP, POS, inventory, invoices, orders, customers, permissions, reports, and automation systems.

> **Building Systems Before Writing Code.**

## Featured Projects

### Tailoring Management System (MTS)

**C# | .NET Framework | Windows Forms | SQL Server | Crystal Reports | 3-Tier Architecture**

A complete desktop system for tailoring shop operations. The system focuses on real business workflows and clean separation between UI, business logic, and data access.

- Order management with delivery and cancellation status
- Customer management and measurement history
- Fabric and inventory tracking with stock deduction
- Purchase invoices, discounts, payments, remaining balance, and refunds
- Login system, users, roles, permissions, and account activation
- Crystal Reports integration for printable operational reports

Repository: [Tailoring Management System MTS](https://github.com/MOGAHEDMOHAMMED/Tailoring-Management-System_MTS)

### Tajer Store Management System

**Flutter | Dart | SQLite | Firebase | GetX | Offline-First**

A mobile-first business system for stores and commercial operations. It supports products, inventory, cashier sales, invoices, debts, expenses, employees, customers, subscriptions, and reports.

- Offline-first local database workflow
- Store dashboard with sales, debts, products, alerts, and reports
- Inventory, cashier, invoices, expenses, and customer tracking
- Built with a structure that separates models, repositories, controllers, and UI logic

Details: [Portfolio](https://mohammedalqhoom.github.io/my-portfolio/)

### Rowad Collect

**PHP | MySQL | PDO | JavaScript | HTML | CSS | PDF Reports**

A responsive web system for billing and collection management.

- Login and dashboard
- Customer and user management
- Invoices, payments, paid/due/overdue tracking
- Search, filters, and PDF financial reports

Details: [Portfolio](https://mohammedalqhoom.github.io/my-portfolio/)
